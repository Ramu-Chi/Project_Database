HOW TO SETUP EVERYTHING:

- Put the project folder outside in D: (D:/Project_Database), IF NOT then the path in load_all_table.sql file need to be changed.
- Create a Database in PgAdmin and load these sql file in the following order:
	+ create_table.sql
	+ create_trigger.sql
	+ load_all_table.sql
	+ create_index.sql
	+ create_functions.sql
